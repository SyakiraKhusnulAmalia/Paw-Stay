# PawStay — Aplikasi Manajemen Penitipan Hewan

Aplikasi web berbasis PHP dan MySQL yang menyediakan layanan 
penitipan hewan peliharaan

## Live Demo
https://pawstay.nfy.fyi

## Fitur

### Pengguna (Customer)
- Dashboard ringkasan penitipan
- Tambah titipan baru (data hewan, jadwal, paket perawatan)
- Lihat status & detail titipan
- Pembayaran (Lunas / DP / Cicilan)
- Profil & ganti password

### Petugas
- Dashboard operasional harian
- Kelola data penitipan
- Catat & perbarui kondisi hewan
- Kelola jadwal penjemputan hewan
- Profil & ganti password

## Teknologi

- PHP
- MySQL
- HTML & CSS
- JavaScript

## Instalasi

1. Clone repository ini
2. Taruh di folder `htdocs` XAMPP
3. Import database di phpMyAdmin
4. Jalankan XAMPP (Apache + MySQL)
5. Buka `localhost/PawStay`
